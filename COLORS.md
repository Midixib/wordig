# UI カラーパレット

wordig の UI で使用する色の仕様です。

---

## ブラック系（テキスト・アイコン・境界）

| 名前       | 16進       | 用途例 |
|------------|------------|--------|
| black-light | `#939393` | 補助テキスト、プレースホルダー、薄いアイコン |
| black-mid   | `#666666` | 本文・サブテキスト |
| black-dark  | `#444444` | 見出し・強調テキスト |

**CSS 変数:** `--black-light`, `--black-mid`, `--black-dark`

---

## ブルー系（メインカラー・背景・アクセント）

| 名前       | 16進       | 用途例 |
|------------|------------|--------|
| blue-50    | `#E8F5FF` | 分析結果カード背景など、最も薄い青背景 |
| blue-100   | `#D4EDFF` | 薄い青背景・ホバー |
| blue-200   | `#B4D7EF` | 境界・区切り |
| blue-300   | `#7DBAE5` | ボーダー・アクセント |
| blue-400   | `#639CC4` | アイコン・リンク |
| blue-500   | `#3986BB` | メインブルー（ボタン・アイコン・枠線） |
| blue-600   | `#2A5E83` | 濃い青テキスト |
| blue-700   | `#134E78` | 最深アクセント |

**CSS 変数:** `--blue-50` ～ `--blue-700`  
（境界用に `--blue-500-30`: メインブルー 30% 透明度）

---

## ホワイト系（背景・カード）

| 名前       | 16進       | 用途例 |
|------------|------------|--------|
| white      | `#FFFFFF` | カード・モーダル・白背景 |
| white-off  | `#E3E3E3` | 区切り線・無効状態 |

**CSS 変数:** `--white`, `--white-off`

---

## 使い方

- **Tailwind（任意値）:** `bg-[var(--blue-50)]` / `text-[var(--black-mid)]`
- **インラインスタイル:** `style={{ backgroundColor: 'var(--blue-500)' }}`
- **JS 定数:** `import { colors } from './src/theme/colors'` で `colors.blue[500]` などを参照

詳細は `src/theme/colors.js` と `index.html` の `:root` を参照してください。
